# Recreating Grooveshark (A Grooveshark Clone)

## Install the dependencies
```bash
./backend
npm install

./frontend
npm install -g @quasar/cli
npm install
```

### Start the Server
```bash
./backend
npm run watch
```
### Start the App
```bash
./frontend
quasar dev
```

### Build the App for Production
```bash
./frontend
quasar build
```
